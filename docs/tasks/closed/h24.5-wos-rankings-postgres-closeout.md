# H24.5 WoS Rankings API Postgres Closeout

Date: 2026-03-13  
Status: historical archived H24 closeout note for `/api/rankings/wos`.

## Summary

`GET /api/rankings/wos` is now a PostgreSQL-backed runtime read surface.

The route name is intentionally unchanged for backward compatibility, but the steady-state read model is `reporting_read.wos_ranking_view`, not Mongo `wos.ranking_view`.

## Steady-State Runtime Decision

- Public API route remains `GET /api/rankings/wos`.
- Runtime query path is:
  - `WosRankingApiController`
  - `WosRankingQueryService`
  - `PostgresWosRankingReadPort`
- Runtime read source is `reporting_read.wos_ranking_view`.
- Runtime Mongo query fallback for this API is intentionally removed.

Mongo remains authoritative for canonical WoS ingest/projection source data, but not for live `/api/rankings/wos` API reads.

## Compatibility Note

The `/api/rankings/wos` name is a retained legacy API namespace.

This should not be interpreted as meaning:
- the route is deprecated,
- the route is Mongo-backed,
- or the route should be renamed as part of normal cleanup.

Future cleanup work should treat the current route name as intentionally stable and the current storage authority as PostgreSQL.
