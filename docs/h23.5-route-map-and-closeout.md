## 3. Removed MVC Route Debt in H25

The following legacy MVC families are removed and must not be used by templates/scripts/docs/tests:

- `/rankings/*`
- `/scholardex/forums*` (public/shared aliases)
- `/core*` (legacy alias family outside `/core/rankings*`)
- `/urap*`
- `/admin/rankings/*` duplicate read GET aliases
- `/user/activityInstances*`
- `/user/individualReports*`
- `/user/publications/scopus_tasks`
- `/user/tasks/scopus/updateCitations`
- `/user/publications/exportCNFIS2025`
- `/user/export/cnfis`
- `/user/rankings/{id}`
- `/admin/scopus/**` MVC compatibility aliases

## 4. API Contract Note

H25 MVC consolidation did not rename `/api/**` contracts.

Notably, `/api/scopus/**` and `/api/rankings/wos` remain stable API namespaces.

## 5. Verification Entry Points

Use the H25/H23 guardrails and route-focused tests when touching route ownership:

- `npm run verify-h25-route-guardrails`
- `npm run verify-h23-ui-guardrails`
- `npm run verify-template-assets`
- targeted MVC/security contract tests for touched route families
