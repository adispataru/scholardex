# H25.1 Canonical Route and Ownership Contract

Date: 2026-03-13  
Status: decision-locked contract for H25 shared MVC route consolidation.

## 1. Summary

This document is the H25.1 source of truth for canonical MVC route ownership.

Locked direction:

- shared authenticated entity reads use entity-first routes,
- user-owned surfaces remain under `/user/*`,
- admin-only management tools remain under `/admin/*`,
- duplicate read route families are compatibility-only during migration and removed early after caller cutover.

No `/api/**` route contract changes are part of H25.1.

## 2. Canonical Shared Authenticated Route Family

The canonical shared read routes are:

- `/forums`
- `/forums/{id}`
- `/wos/categories`
- `/wos/categories/{key}`
- `/core/rankings`
- `/core/rankings/{id}`
- `/universities`
- `/universities/{id}`
- `/events`

Ownership: `shared-authenticated`.

Access intent: authenticated users across roles can read these pages. Admin-only operations remain separately role-gated.

## 3. User-Owned Route Family

The user-owned route family remains under `/user/*`.

Locked user-owned surfaces:

- `/user/dashboard`
- `/user/publications`
- `/user/activities` (target canonical name; current `/user/activityInstances` migrated in H25.3)

Ownership: `user-owned`.

## 4. Admin-Only Management Surfaces (Non-Goals)

H25 does not move admin-only management pages out of `/admin/*`.

Examples that remain under admin ownership:

- `/admin/initialization/**`
- `/admin/conflicts/**`
- `/admin/source-links/**`
- admin edit/maintenance actions for forums/authors/affiliations/publications
- admin catalog/report management routes

Ownership: `admin-only-management`.

## 6. Explicit Non-Goals for H25.1

- No API namespace renaming.
- No admin-only operation route redesign.
- No full duplicate template deletion in this subtask.
- No `/user/activityInstances` rename implementation in this subtask (owned by H25.3).

## 7. Gate for H25.2

H25.2 implementation must treat this file as the canonical contract and align route wiring, templates, tests, and docs to this mapping.
